# Docker

## Run Docker dev env

### API
- Create .env files from .env.docker sample
- `yarn install`

### WEB
- Create .env files from .env.docker sample
- `docker-compose up -d`

### Dozzle
- Visit http://localhost:9999 to watch container logs

### Docker scripts
- Use docker/cli.sh to run api cli commands in docker: `docker/cli.sh --help`
