# Root app

### How to run

#### Requirements
- node v.12
- TypeScript v.3.8.x
- PostgreSQL 12.x

#### API
- `cd ./api`
- `yarn`
- `yarn global add nodemon`
- create `.env` file from `.env.example`
- `nodemon`

#### Debug
- Set `logging: true` in `db.ts` to see in console all SQL queries sent to PostgreSQL server
